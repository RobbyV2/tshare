TShare

Terminal sharing through web links.

Installation

Download pre-built packages from releases or build from source: 

cargo install --path .

Usage

Start servers:

tunnel-server &
web-server &

Share terminal:

tshare connect

Architecture

  • tshare: CLI client
  • tunnel-server: WebSocket relay (ports 8080/8081)
  • web-server: Web interface (port 3000)

Source: github.com/RobbyV2/tshare
